# Portfolio

A modern personal portfolio website built with Next.js, React, Tailwind CSS, and Three.js.

This project showcases a developer portfolio with interactive 3D effects, animated UI, and production-ready tooling. It includes Sentry support and a simple API route example for monitoring and error handling.

---

## Demo

This project is ready to deploy to Vercel or any host that supports Next.js.

---

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- `next-themes` for theme handling
- `@sentry/nextjs` for error monitoring
- Vercel deployment support

---

## Features

- Responsive portfolio landing page
- 3D globe and visual canvas effects
- Animated UI sections and transitions
- Modular component structure
- Tailwind CSS styling
- Sentry integration for client/server monitoring
- Example API route at `/api/sentry-example-api`
- Static generation for pages and optimized production build
